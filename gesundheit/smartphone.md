# Smartphone

## Umgang mit dem Smartphone

Folgende Tipps führen primär zur gesteigerten inneren Ruhe.

Die Auswirkungen von innerer Ruhe sind verminderter Blutdruck, gesünderer Ruhepuls und gesteigerte Resilienz.

| Tätigkeit | Tipp |
| --- | --- |
| Duschen | Lege das Smartphone in einen anderen Raum |
| Spazierengehen | Lass das Smartphone zu Hause |
| Arbeiten | Leg das Smartphone in einen anderen Raum |
| Schlafen | Ged das Smartphone vor die Tür oder schalte den Flugmodus an |
| Filme/Musik/Medien konsumieren | Schalte den Flugmodus an |
